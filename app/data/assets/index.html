Как это работает?

Поиск репозиториев по предметной области. Часть 1

Поиск предметных областей начинается со сбора данных из описаний репозиториев. Данный, которые используются в анализе - блок About и блок Topics (тэги). Система выполняе усечение количества ключевых слов из блоков, за счет выделения самых значимых слов.

Поиск репозиториев по предметной области. Часть 2

После выделеения ключевых слов, формируется словарь и для каждого репозитория составляется вектор.

Поиск репозиториев по предметной области. Часть 3

Из полученных векторов, построенных на ключевых словах, выполняется расчет косинусного расстояния для каждого с каждым.

Поиск похожих репозиториев на основе сравнения Issue. Часть 1.

Для вашего репозитория выполняется сравнение близости текста каждого Issue с каждым Issue похожих по предметной области репозиториев. Сравнение схожести текстов состоит из двух этапов: сравненеие текстов блока Title, сравнение блоков Body. На первом этапе сравнивается процентное соотношения общих слов в блоке Title. Если число пересечений менее 70%, то считается что такие Issue не являются похожими.

Поиск похожих репозиториев на основе сравнения Issue. Часть 2.

Формируется общий словарь для пар Title и Body. После строятся вектора и выполнется рассчет схожести в несколько этапов: Первый этап - считаются проценты содержимого в каждом векторе относительного словарей. Второй этап - считаются косинусные расстояния. Третий этап - коэффициенты суммируются и получется итоговая оценка схожести.

Поиск похожих репозиториев на основе сравнения Issue. Часть 3.

Формируется оценка схожести репозиториев по формуле: K / (M x N), где K - количество пар похожих Issue, M и N - количество Issue двух репозиториев соответственно. Такая оценка называется - Рангом.

Заполните форму и получите ТОП 10 репозиториев, похожих на ваш репозиторий.

Если репозиторий, который вы укажите в форме уже проиндексирован в базе данных, то вы будете переброшены на страницу с ТОП 10, иначе в отложенном режиме, спустя некоторое время, вам на почту будет отправлено письмо содержащее ссылку на загрузку результата.
